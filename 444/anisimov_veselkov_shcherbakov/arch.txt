Архитектурные требования
● Система должна быть реализована в виде приложения для операционной системы Windows.
● Серверная часть состоит из SQL сервера.
● Система должна использовать в качестве БД MySQL.

Технологии:
● Приложение должно быть разработано на языке C#.
● Приложение связывается с БД посредством Oracle MySQL C# Connector.
